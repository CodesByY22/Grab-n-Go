# 🍱 Grab-n-Go | College Cafeteria App

**Grab-n-Go** is a website made to help students order food from the college cafeteria without waiting in line. It connects students and shop owners in real-time.

---

## 🌟 Main Features

### 🎓 For Students
- **Modern Look**: A clean and easy-to-use design that works on both mobile and laptop.
- **Easy Browsing**: See all campus shops and check if they are "Open" or "Closed" before you walk there.
- **Simple Ordering**: Add food items to your cart, see the total price, and place your order.
- **Live Tracking**: Watch your order status change from "Preparing" to "Ready" on your screen.

### 👨‍🍳 For Shop Owners (Vendors)
- **Order Dashboard**: See new orders the moment they come in.
- **Menu Management**: Easily add new dishes or hide items that are out of stock.
- **Live Updates**: Click a button to let the student know their food is ready for pickup.
- **Shop Status**: Open or Close the store with one simple switch.

---

## 🛠️ Built With (MERN Stack)

- **Frontend**: React (for the website pages).
- **Backend**: Node.js and Express (for the server).
- **Database**: MongoDB (to save all the menu and user data).
- **Real-time Updates**: Socket.io (to send instant notifications).

---

## 🏁 How to Start

### 1. Setup Data
```bash
cd backend
node seeder.js # This will add the demo shops and images to the database
```

### 2. Run the Project
- Open a terminal in `backend` and run `npm run dev`.
- Open another terminal in `frontend` and run `npm run dev`.
